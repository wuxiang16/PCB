# RK809-5 电源树设计

## 1. DC-DC（Buck） vs LDO
### 1.1 为什么需要它们

一块板子只有一个输入电压（如 5V），但不同器件需要不同电压：

* CPU：~1V
* DDR：1.1V
* IO：1.8V / 3.3V

👉 必须“降压” → 用 **Buck / LDO**

### 1.2 DC-DC（Buck）

👉 **高效降压电源**

* 原理：开关 + 电感“搬运能量”
* 特点：

  * ✅ 效率高（省电）
  * ✅ 发热少
  * ❌ 有噪声
* 用途：

  * CPU / GPU / DDR（大电流）

👉 一句话：**高效供电主力**

---

### 1.3 LDO 低噪声线性稳压

* 原理：多余电压 → 变热
* 特点：

  * ✅ 输出干净（低噪声）
  * ❌ 效率低（会发热）
* 用途：

  * 音频 / 摄像头 / 模拟电路

👉 一句话：**精细稳压**

### 1.4 实际用法（重点）

👉 一般组合使用：

```
5V → Buck → 1.8V → LDO → 1.2V
```

👉 目的：

* Buck：先高效降压
* LDO：再降低噪声
